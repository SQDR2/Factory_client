# Factory_client

这个项目为工厂后台管理系统，这是后台的管理
其中模块分为员工、管理员、供应商、产品、订单、权限角色等模块。使用的前端技术为Vue3.0全家桶+TS+ElementUI Plus。
还有一个前台的员工签到小程序（Factory_user）

### 使用技术栈：

Vue3.0 + TS + ElementUI Plus + Axios + dayjs + ant-design-vue + vue-router等

### 安装环境：

npm install

### 运行：

npm run dev

### 打包：

npm run build
